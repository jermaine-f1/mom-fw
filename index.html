🎯 Core Strategy Principles

1. Momentum + Diversification
  • • Screen for ETFs with quality momentum (not just raw return)
  • • Select holdings that are LOW correlation to each other
  • • Spread across countries, sectors, asset classes
  • • Goal: Capture momentum while diversifying risk
2. Many Small Bets
  • • Equal weight or near-equal across positions
  • • No single position > 15%
  • • Target 6-10 positions for diversification
  • • If one fails, portfolio survives
3. Quality Over Raw Return
  • • Same 18% return — take the smoother trend
  • • Prioritize: High Sortino, few down weeks, steady slope
  • • Avoid: Choppy, extended, parabolic moves
  • • Sustainable > Spectacular
4. Flexible Screening
  • • Ranked universe — no rigid buy/sell labels
  • • Use filters to narrow: MA position, Z-Score, min score
  • • Top-ranked ideas surface naturally
  • • Judgment + data, not autopilot
